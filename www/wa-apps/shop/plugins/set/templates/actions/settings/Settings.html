Настройка плагина

Плагин позволяет создавать и управлять акционными комплектами товаров.

Акционный комплект состоит из основого товара без скидки, к которому назначаются товары со скидкой.

Управление комплектами осуществляется в разделе редактора товара во вкладке "Комплекты". Для каждого артикула товара можно создавать отдельные комплекты. В комплект товары добавляются с точностью до артикула.

Вывод комплекта (или комплектов) товара осуществляется через хук frontend_product.block на странице товара или с помощью хелпера {literal}{shopSetPlugin::set($product)}{/literal}, где $product - переменная шаблона страницы товара.

В корзину комплект добавляется как отдельные товары со скидкой. Если указаны jQuery-селекторы элементов корзины в настройке плагина, то в корзине указывается информация о комплектах для данного товара с возможность изменять количество комплекта.

Для настройки обновления корзины без перезагрузки страницы при добавлении комплекта в корзину на странице товара внесите изменения в метод afterAddCompect в блоке head.

Значения настроек по умолчанию даны для темы default.
